Concept overview

How this site meets the rubric

Minimum required elements

  • Customer service chat UI (Chat)
  • Customer self-service order status UI (Order status)
  • Shopping cart UI (Cart)
  • Order calculation and confirmation UI (Totals + Confirmation)
  • Online payment fields UI (Checkout)
  • Categorized images and product descriptions (Shop)

Common elements

  • Consistent navigation, footer, and breadcrumbs
  • Limited color palette (primary/secondary/accent + neutrals)
  • Image library in /images with alt text
  • Responsive layout using media queries and CSS grid
Scope: UI-only prototype. No server, database, or real payment processing.
